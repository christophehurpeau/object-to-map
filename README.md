# object2map [![NPM version][npm-image]][npm-url]

[![Build Status][build-status-image]][build-status-url] [![Coverage][coverage-image]][coverage-url]

## Quick example

```js
var object2map = require('object2map');
console.log(object2map({ test: true }));
```
